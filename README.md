# Zomato Food Delivery Clone

A modern, responsive food delivery app built with React, Vite, and Tailwind CSS. This project replicates the core functionality of Zomato including restaurant browsing, menu viewing, cart management, and order tracking.

## Features

- **Restaurant Discovery**: Browse restaurants with cuisine filters
- **Menu Navigation**: Detailed restaurant pages with interactive menus
- **Shopping Cart**: Add/remove items with quantity management
- **Order Tracking**: Real-time order status simulation
- **User Profile**: Order history and account management
- **Responsive Design**: Mobile-first approach with desktop optimization

## Technologies Used

- **React 18** - Frontend framework
- **Vite** - Build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icon library
- **JavaScript** - Programming language

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd zomato-da
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
